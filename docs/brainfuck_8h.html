◆ build_loops()

const loop * build_loops ( const char *  __input)

Construit le tableau des boucles.

Analyse le programme Brainfuck passé en paramètre pour construire un loop qui représente les boucles du programme. Celui-ci sera ensuite utilisé lors de l'exécution des instructions.

Paramètres
__inputle programme Brainfuck à analyser
Renvoie
un loop qui représente les boucles du programme
Remarques
Pour des raisons d'implémentation, avoir plus de 255 boucles imbriquées les unes dans les autres n'est pas possible. Dans le cas où cette condition n'est pas respectée, le comportement est indéfini.
